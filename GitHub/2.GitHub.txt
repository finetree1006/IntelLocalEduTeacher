2.GitHub.txt


1)깃허브란?
Git을 사용할 때 소스코드를 올려놓는 서버가 필요한데, 
이 서버를 제공해 주는 서비스

2)기능
.소스코드 관리
.프로젝트 설명서
.커뮤니티

.깃허브 이용비용
-오픈소스 프로젝트: 무료
-사용 private 프로젝트: 유료

*필요 이유
-개인적으로 서버를 구현하는 것은 매우 번거로움
-GitHub에 프로젝트만 생성하면 누구나 쉽게 Git의 중앙 서버로
Github를 사용할 수 있음
-각 개발자들의 소스코드의 상태를 관리.저장해 주는 저장소 역할
-안전하게 프로젝트를 진행할 수 있음
-다른 이들이 만든 프로젝트들을 쉽게 찾고 사용할 수 있음
-다른 이용자와 질문과 답변 가능
-통계 기능 사용 가능


이제는 실제 사이트 사용법 좀 배우는 거 할 차례
